AI Number Recognizer






This is an AI Machine Learning Project that can identify any digit you draw!

It works by training a CNN (Convolutional Neural Network) on thousands of handwritten digits. Through this process, the model learns the underlying patterns and creates the weights needed to recognize new digits with a small margin of error.

Since training machine learning models can take a long time, I added GPU support to speed things up. I also implemented save/load functionality with .pth files, so you don’t have to retrain the model every time you run the program. Loading the model still takes some time, but once it’s up and running, it can identify numbers almost instantly. On a server running 24/7, this would work seamlessly.

To make the data easier to process, machine learning models often perform best with grayscaled and pixelized images. I also made it center and resize the image to make it look more like the training data. These steps are necessary for the model to be able to correctly interpret the images and make the best prediction.

I find it very cool that thanks to the advancement of AI libraries, I can quickly create and run a program on my PC that demonstrates real intelligence.
